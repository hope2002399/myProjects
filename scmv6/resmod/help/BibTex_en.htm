Method 1：Export BibTeX file from EndNote.

 

1.      Select the target record(s) you would like to export into a file.  Clic k ”FileàExport” to export the record(s).

 

2.      Select “BibTex Export” as default output style and click the button [Save] to save the file into the local machine .

 

PS： Please refer to the second method for award, patent and others file.

 

Method 2：Download BibTeX file from google scholar.

 

1.      Search the target publications from “google scholar”. Click the “reference” button.

 

2.      Click the “BibTex” button to find the “BibTex” file content. ( If you want to upload EndNote file, please select the “EndNote” button).

 

3.      Create a new TXT file. Copy the “BibTex” file content mentioned in the step 2 and paste them into the new TXT. Then upload the new TXT file.
